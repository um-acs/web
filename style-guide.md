# HPC Portal Brand Style Guide (University of Miami Inspired)

## Brand Direction
This UI follows a University of Miami–aligned visual language:
- Bold **orange + green** accents
- Clean, modern academic layout
- Clear typography hierarchy
- Accessible contrast and generous spacing

## Color Tokens
- `--miami-orange: #F47321`
- `--miami-green: #005030`
- `--ink: #12332A`
- `--text: #1D2B28`
- `--muted: #5B6662`
- `--bg: #F7F9F8`
- `--surface: #FFFFFF`
- `--border: #DDE4E1`

## Typography
- Font stack: `Inter, Segoe UI, Roboto, Helvetica, Arial, sans-serif`
- Heading style: strong, high contrast
- Body style: readable, moderate line-height

## Layout
- Mobile-first responsive sections
- Max content width: ~1200px
- Card-based information architecture
- Visible CTA hierarchy (primary orange, secondary white/green border)

## Components
- Sticky top nav with white links on dark green
- Hero with dark overlay + key CTAs
- Resource cards with icon, summary, action
- Soft shadows, subtle hover elevation

## Voice & Tone
- Professional and concise
- Clear action-oriented labels
- Student/faculty/staff friendly

## Future Additions
- Add official UM typography if licensed/approved
- Add accessibility audit checks (color contrast + keyboard)
- Define spacing scale + component variants (buttons/cards/badges)
